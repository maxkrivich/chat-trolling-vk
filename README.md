# Chat Trolling VK
> If you like trolling people in VK, use this script! =)

***

### Usage
```sh
$ ./chat_prolling.py -e <email> -p <password> -h <help>
```
###### stop: Ctrl + C

### Requirements

* requests
* HTMLParser
* getpass

### Installing
```sh
$ pip install requests
$ pip install HTMLParser
$ pip install getpass
```
